### Why this repo?

Small tools for better life

### How to use

#### requirements

node environment

#### install

```javascript
npm install @talentan/tools -g
```

#### examples

```javascript
// 批量重命名文件后缀
pls reSuffix doc txt // 把当前文件夹下(不包括子文件夹)的 *.doc重命名成 *.txt
pls reSuffix doc txt -a // 把文件夹下(包括子文件夹)的 *.doc重命名成 *.txt

```
